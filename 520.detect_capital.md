#### 520.检测大写字母
[题目链接](https://leetcode-cn.com/problems/detect-capital/)
> 给定一个单词，你需要判断单词的大写使用是否正确。
>
> 我们定义，在以下情况时，单词的大写用法是正确的：
>
> 1. 全部字母都是大写，比如"USA"。
> 2. 单词中所有字母都不是大写，比如"leetcode"。
> 3. 如果单词不只含有一个字母，只有首字母大写， 比如 "Google"。
>
> 否则，我们定义这个单词没有正确使用大写字母。
>
> **示例 1:**
>
> ```
> 输入: "USA"
> 输出: True
> ```
>
> **示例 2:**
>
> ```
> 输入: "FlaG"
> 输出: False
> ```
>
> **注意:** 输入是由大写和小写拉丁字母组成的非空单词。

**简单思路**

对题目所说的三种情况进行判断即可。执行时间56ms

```python
class Solution:
    def detectCapitalUse(self, word):
        """
        :type word: str
        :rtype: bool
        """
        
        allCap, allNotCap = True, True
        lastCap = 0
        
        for idx, ch in enumerate(word):
            if ch >= 'a' and ch <= 'z':
                allCap = False
            else:
                allNotCap = False
                lastCap = idx
        
        if allCap is True or allNotCap is True:
            return True
        else:
            return True if lastCap == 0 else False
```

```python
class Solution:
    def detectCapitalUse(self, word):
        """
        :type word: str
        :rtype: bool
        """
        
        return word.upper() == word or word.lower() == word or word.title() == word
```
