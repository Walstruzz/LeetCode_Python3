#### 171.Excel表列序号
[题目链接](https://leetcode-cn.com/problems/excel-sheet-column-number/)
> 给定一个Excel表格中的列名称，返回其相应的列序号。
>
> 例如，
>
> ```
>     A -> 1
>     B -> 2
>     C -> 3
>     ...
>     Z -> 26
>     AA -> 27
>     AB -> 28 
>     ...
> ```
>
> **示例 1:**
>
> ```
> 输入: "A"
> 输出: 1
> ```
>
> **示例 2:**
>
> ```
> 输入: "AB"
> 输出: 28
> ```
>
> **示例 3:**
>
> ```
> 输入: "ZY"
> 输出: 701
> ```

**简单思路**

其实就是26进制转10进制。时间复杂度$O(n)$，空间复杂度$O(1)$，执行时间100ms

```python
class Solution:
    def titleToNumber(self, s: str) -> int:
        
        res = 0
        
        for ch in s:
            res = res * 26 + (ord(ch) - ord("A") + 1)
        
        return res
```
